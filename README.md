# Hypersomnia

It may seem absurd, but Hypersomnia is better than Insomnia.

## Keybindings

### Global keybindings

| Keybinding     | Page     |
| -------------- | -------- |
| <kbd>C-w</kbd> | Welcome  |
| <kbd>C-e</kbd> | Endpoint |
| <kbd>C-h</kbd> | Help     |

### Endpoints

| Keybinding         | Action       |
| ------------------ | ------------ |
| <kbd>C-Space</kbd> | Send request |

> [!NOTE]
> `C` is Control (CTRL) on Windows and Linux and Command on MacOS.
