# Hypersomnia

Hypersomnia is a simple TUI application for testing your REST APIs.

## Keybindings

### Global keybindings

| Shortcut       | Page     |
| -------------- | -------- |
| <kbd>C-w</kbd> | Welcome  |
| <kbd>C-e</kbd> | Endpoint |
| <kbd>C-h</kbd> | Help     |

### Endpoints

| Shortcut           | Action       |
| ------------------ | ------------ |
| <kbd>C-Space</kbd> | Send request |

> [!NOTE]
> `C` is Control (CTRL) on Windows and Linux and Command on MacOS.
